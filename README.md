# Comic Reader

A full in-browser comic reader for BackIssue. Open any owned issue and read
it right there — **paged**, **double-page**, and **webtoon** modes, reading
direction and fit options, keyboard/touch/tap navigation, and a keep-awake
toggle. Progress is tracked **per user**: resume where you left off,
bookmarks, a "mark read at N%" threshold, per-series reading profiles, home
reading shelves (Continue reading, Read later, and more), and reading stats
on your profile.

## Install

One click from **Sidebar → Plugins** in BackIssue, or drop this folder into
the app's `plugins/` directory and restart.

## Setup

Nothing required — a **Read** button appears on owned issues once the plugin
loads. Options live in:

- **Settings → Library** — use the file's first page or ComicVine art as
  issue covers.
- **Your profile** — personal reading defaults (mode, direction, fit,
  read-at-%).

Reading is governed by its own grantable permission, so you can decide per
role who gets the reader. Installing it also unlocks page streaming and
personal shelves in the [OPDS plugin](https://backissue.app/opds).

Full guide: <https://backissue.app/reading>
